= CSI volume cloning
include::../includes/common-attributes.adoc[]
:context: persistent-storage-csi-cloning


Volume cloning duplicates an existing persistent volume to help protect against data loss in {product-title}. This feature is only available with supported Container Storage Interface (CSI) drivers. You should be familiar with xref:persistent-volumes_understanding-persistent-storage[persistent volumes] before you provision a CSI volume clone.
