= Monitoring project and application metrics using the Developer perspective
include::includes/common-attributes.adoc[]
:context: monitoring-project-and-application-metrics-using-developer-perspective



The *Observe* view in the *Developer* perspective provides options to monitor your project or application metrics, such as CPU, memory, and bandwidth usage, and network related information.

[id="prerequisites_odc-monitoring-project-and-application-metrics-using-developer-perspective"]
== Prerequisites

* You have xref:odc-creating-applications-using-developer-perspective[created and deployed applications on {product-title}].
* You have link:https://access.redhat.com/documentation/en-us/openshift_container_platform/4.14/html-single/web_console/#web-console[logged in to the web console] and have switched to link:https://access.redhat.com/documentation/en-us/openshift_container_platform/4.14/html-single/web_console/#about-developer-perspective_web-console-overview[the *Developer* perspective].
