= Installing Service Binding Operator
include::../includes/common-attributes.adoc[]
include::../includes/servicebinding-document-attributes.adoc[]
:context: installing-sbo


[role="_abstract"]
This guide walks cluster administrators through the process of installing the {servicebinding-title} to an {product-title} cluster.

You can install {servicebinding-title} on {product-title} 4.7 and later.

[discrete]
== Prerequisites

* You have access to an {product-title} cluster using an account with `cluster-admin` permissions.
* Your cluster has the link:https://access.redhat.com/documentation/en-us/openshift_container_platform/4.14/html-single/postinstallation_configuration/#enabling-cluster-capabilities[Marketplace capability] enabled or the Red Hat Operator catalog source configured manually.
