ifndef::openshift-origin[]
When opening a support case, it is helpful to provide debugging
information about your cluster to Red Hat Support.

It is recommended to provide:

* xref:support_gathering_data_gathering-cluster-data[Data gathered using the `oc adm must-gather` command]
* The  xref:support-get-cluster-id_gathering-cluster-data[unique cluster ID]
endif::openshift-origin[]
